# Tinyseed

## TinySeed is a fork of Binary Holding's Tenderseed, which is a fork of Polychain's Tenderseed

To seed all 60 chains in cosmos:


```bash
git clone https://github.com/notional-labs/tinyseed
go mod tidy
go install .
tinyseed
```
